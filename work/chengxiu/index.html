a. Hypothesis:

How do emoji designs and applications on different digital platforms reflect and create platform-specific digital cultures?

c. Computational methods:

  1. Data Visualization: Using React and D3.js to create an interactive timeline and comparison of emoji evolution.
  2. python data crawling: study the UI/UX design patterns in emoji implementation.

d. Design methodology:

  • Analytical: examining how different platforms have implemented and adapted emoji systems.
  • Critical: Questioning the limitations of standardized emoji sets.
  • Experimental: exploring the similarities and differences in the use of visual emoji.

e. Precedents

  1. Emojipedia's annual emoji statistics

    Provides comprehensive usage data, but focuses primarily on Unicode standards.

  2. Uncanny Values - Process Studio

    The Uncanny Values project inspired this research to think about digital emotional expression, but unlike its art installation presentation, this project adopts a computational design methodology to explore the evolutionary patterns and usage modes of emoji as a cross-cultural digital language through data analysis and interaction design, aiming to provide a deeper understanding and practical tools for digital communication.
